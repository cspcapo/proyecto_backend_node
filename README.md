# proyecto_backend_node
Destinado a la entrega del trabajo practico final del curso de Back end Node JS.

Para inicializar la base de datos usar npm run seeder

Para crear un item se deben colocar estos datos obligatorios : 
{
    "mark": "",
    "name": "",
    "price": "",
    "stock": "", 
}
Ejemplo a para crear :

{
    "name": "Camara Ptz WiFi 2mp interior",
    "price": "50000",
    "stock": "15", 
    "mark": "tapo"
}
